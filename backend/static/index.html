Steganography API Online

Welcome to the Steganography API. This service provides endpoints for hiding and retrieving data in various media types.

Available Endpoints

  • Image Steganography:
    • POST /encode/image - Hide a message in an image
    • POST /decode/image - Extract a message from an image
  • Audio Steganography:
    • POST /audio/encode - Hide a message in an audio file
    • POST /audio/decode - Extract a message from an audio file
  • File-in-Image Steganography:
    • POST /image-file/encode - Hide a file in an image
    • POST /image-file/decode - Extract a file from an image
  • Video Steganography:
    • POST /video/encode - Hide a message in a video
    • POST /video/decode - Extract a message from a video
